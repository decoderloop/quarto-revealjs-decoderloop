# Decoder Loop's Quarto Reveal.js presentation theme

A theme for [Quarto](https://quarto.org/) Reveal.js presentations, based on [grantmcdermott/quarto-revealjs-clean](https://github.com/grantmcdermott/quarto-revealjs-clean/).

## Use

Adding the theme to an existing directory:

```bash
quarto install extension decoderloop/quarto-revealjs-decoderloop
```

Starting from scratch with a content template:

```bash
quarto use template decoderloop/quarto-revealjs-decoderloop
```
